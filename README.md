# 🚂 Advanced 3D Railway Crossing Animation using Python OpenGL

## 📌 Project Overview

This project is an advanced **3D Railway Crossing Simulation** developed using **Python OpenGL (PyOpenGL + GLUT)** for the Computer Graphics Lab course.

The project demonstrates various Computer Graphics concepts including:

- Blending
- Shading
- Fog Effect
- 3D Scene Rendering
- Camera Movement
- Animation
- Event Handling
- Translation
- Rotation
- Push/Pop Matrix
- Primitive Drawing
- Algorithm-Based Pixel Drawing Concepts

The scene contains a moving train, animated railway gate, smoke effect, buildings, trees, road, railway tracks, sky, clouds, and interactive camera control.

---

# 🎯 Project Objectives

The main goal of this project is to implement core Computer Graphics concepts in a realistic and interactive 3D environment using OpenGL.

This project also demonstrates practical implementation of:
